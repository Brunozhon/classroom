JavaScript Object Notation

JSON can let you store data like this:

{"languges":[
  {
    "name":"HTML",
    "file-extiostions":[
      ".html",
      ".htm",
      ".inc",
      ".xht",
      ".xhtml"
    ]
  },
  {
    "name":"CSS"
    "file-extiostions": [
      ".css"
    ]
  }
]}

Then, you can JSON.parse() the result. In the code above, the possible code results are:

  • varname.languges[0] results in [object Object]
  • varname.languges[0].file-extiostions results in .html,.htm,.inc,.xht,.xhtml
  • varname.languges[2] results in undefined
